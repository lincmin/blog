JavaScript 语言的下一代标准 ES6

发表于 2016-12-18 分类于 JavaScript

概念简介

ES6 是什么

ES6 全称为 ECMAScript 6.0，是 JavaScript 语言的下一代标准，已经在 2015 年 6 月正式发布。它的目标是使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。ECMAScript 和 JavaScript 的关系是，前者是后者的规格，后者是前者的一种实现。
ES6 既是一个历史名词，也是一个泛指，含义是 5.1 版以后的 JavaScript 的下一代标准，涵盖了 ES2015、ES2016、ES2017 等等。

Babel 是什么

Babel 是一个广泛使用的 ES6 转码器，可以将 ES6 代码转为 ES5 代码，从而在现有环境执行。这意味着，你可以用 ES6 的方式编写程序，又不用担心现有环境是否支持。

ES6 常用特性

const 和 let

const 声明一个只读的常量。一旦声明，常量的值就不能改变。所以声明后必须立即初始化。let 声明的是变量

  • 块级作用域，区块之间的代码互不影响
  • 不存在变量提升，遵循先声明后使用的原则

模板字符串

模板字符串（template string）是增强版的字符串，用反引号（`）标识。它可以当作普通字符串使用，也可以用来定义多行字符串，或者在字符串中嵌入变量。

箭头函数

函数的快捷写法，不需要通过 function 关键字创建函数，并且还可以省略 return 关键字。同时，箭头函数还会继承当前上下文的 this 关键字。

模块的 Import 和 Export

import 用于引入模块，export 用于导出模块。

析构赋值

Promises 对象

Promise 对象代表一个异步操作，可以解决回调函数嵌套的问题。有三种状态：pending（进行中）、fulfilled（已成功）和 rejected（已失败）。

Generators

Generator 最大特点就是可以交出函数的执行权，通过 yield 关键字实现暂停功能。可以使异步逻辑通过同步的方式组织起来。一个简单的 Generator 函数示例.
