文本数据交换格式JSON

发表于 2016-06-26 分类于 JavaScript

什么是 JSON 格式

JSON 全名是 JavaScript Object Notation(JavaScript 对象表示法)，是一种用于存储或者数据交换的文本信息格式。相比同样是用于数据交换的 XML，JSON 有更小、更快，更易解析符合 JavaScript 原生语法的优点。

JSON 语法

JSON 语法规则

  • 数据用 key/value（键/值）对表示
  • 每个 key/value 对使用逗号,分割
  • key 和 value 中使用冒号:分割
  • key 必须是字符串，value 可以是合法的 JSON 数据类型（字符串, 数字, 对象, 数组, 布尔值或 null）
  • 大括号{}保存对象
  • 中括号[]保存数组
  • 字符串必须使用双引号表示，不能使用单引号
  • 对象的键名必须放在双引号里面
  • 数组或对象最后一个成员的后面，不能加逗号

JSON 值

  • 支持原始类型的值只有四种：字符串、数值（必须以十进制表示）、布尔值和 null（不能使用 NaN, Infinity, -Infinity 和 undefined），复合类型的值只能是数组或对象，不能是函数、正则表达式对象、日期对象。
  • 数组（在中括号中）
  • 对象（在大括号中)，对象可以包含多个 key/value（键/值）对
  • 数字（整数或浮点数）
  • 字符串（在双引号中）
  • 逻辑值（true 或 false）

JSON 对象

访问对象值

  • 使用点号.来访问对象的值
  • 使用中括号[]来访问对象的值
  • 修改对象值
  • 删除对象属性
  • 使用 for-in 来循环对象的属性和值

JSON.parse()

JSON 对象是 JavaScript 的原生对象，JSON.parse()是其中一个常用的静态方法，它的作用是将字符串数据转换为 JavaScript 对象

语法

JSON.parse(text[, reviver])

  • text 参数是要被解析成 JavaScript 值的字符串
  • reviver 参数可选，一个转换结果的函数， 将为对象的每个成员调用此函数

异常处理

如果字符串不符合 JSON 格式，就会报错，为了处理解析错误，可以将 JSON.parse 方法放在 try…catch 代码块中

JSON.stringify()

JSON.stringify()是 JSON 对象的另一个静态方法，JSON.stringify() 方法将 JavaScript 对象转换为字符串

语法

JSON.stringify(value[, replacer[, space]])

  • value 参数是一个有效的 JSON 对象
  • replacer 参数可选，用于转换结果的函数或数组
  • space 参数可选，文本添加缩进、空格和换行符
