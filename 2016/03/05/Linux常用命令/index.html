Linux常用命令

发表于 2016-03-05 分类于 Linux

文件管理

查看目录中文件列表

  • -al显示文件属性以及文件所属的用户和组（包括隐藏文件）

创建目录

  • -p 确保父目录名称存在，不存在的就建一个

创建文件

  • 如果文件已存在，则修改文件或者目录的时间属性为当前系统时间

删除文件或者目录

  • -r 将目录及以下之档案亦逐一删除，-f 即使原档案属性设为只读，亦直接删除，无需逐一确认

复制文件或目录

  • -r 复制该目录下所有的子目录和文件

移动文件

  • -v 显示命令执行详情

重命名文件或文件夹

  • -v 显示命令执行详情

创建同步链接

  • -s 创建软链接(符号链接)，以路径的形式存在。类似于 Windows 操作系统中的快捷方式

更改文件属主和文件属组

修改文件的基本权限

  • -R 进行递归(recursive)的持续变更，即连同次目录下的所有文件都会变更

1 . 数字修改，r:4，w:2，x:1

2 . 字符修改 ，u（user 属主），g（group 属组）， o（others 其他），a（ all 全部），+(加入)，-(除去)，=(设定)

系统管理

查看系统环境信息

输出系统变量值，查看 shell

获取操作系统的信息

查询命令历史

  • | grep过滤关键字

实时显示进程 (process) 的动态

显示当前进程 (process) 的状态

  • aux显示所有包含其他使用者的进程

删除执行中的程序或工作

  • -9彻底杀死进程

建立用户帐号

  • -m自动建立用户的登入目录/home/userOne

更改使用者的密码

删除用户帐号

  • -r 删除用户登入目录以及目录中所有文件

切换用户

网路通讯

查看机器网络套接字连接情况(端口号)

  • u显示 UDP 传输协议的连线状况
  • n直接使用 IP 地址，而不通过域名服务器
  • t显示 TCP 传输协议的连线状况
  • p显示正在使用 Socket 的程序识别码和程序名称
  • l显示监控中的服务器的 Socket

查看 80 端口使用情况

检测是否与主机连通

显示网络设备信息

启动关闭指定网卡

配置 IP 地址

程序管理

查找命令所在路径
