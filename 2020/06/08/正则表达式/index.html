正则表达式

发表于 2020-06-08 分类于 JavaScript

使用正则表达式处理规则复杂的字符串查找、替换需求,按照某种规则去匹配符合条件的字符串;
Regexper

RegExp对象

javascript 通过内置对象RegExp支持正则表达式
有两种方法实例化RegExp对象

字面量

\b 表示字符边界;
g 边是全局搜索匹配;

构造函数

修饰符

g:global 全文搜索，如果不添加，搜索到第一个匹配停止;
i:ignore case 忽略大小写，默认大小写敏感;
m:multiple lines多行搜索;

元字符

正则表达式由两种基本字符类型组成;
原义文本字符，如’abc’;
元字符，有特殊含义的非字母字符。如\b表示字符边界

字符类

使用元字符[]构建一个简单的类;
所谓类是指符合某些特性的对象，一个泛指，而不是特指某个字符;
表达式[abc]把字符a 或b 或c归为一类，表达式可以匹配这类的字符;

字符类取反

使用元字符^创建反向类/负向类;
反向类就是不属于某类的内容;
表达式[^abc]表示不是字符a或b或c的内容

范围类

可以使用[a-z]来连接两个字符表示从a到z的任意字符，这是个闭区间，包含a和z本身;
在[]组成的类内部可以连写[a-zA-Z],表示包含大小写字母;

JS预定义类及边界

预定义类

字符 等价类 含义
. [^\r\n] 除了回车符和换行符之外的所有字符
\d [0-9] 数字字符
\D [^0-9] 非数字字符
\s [\t\n\x0B\f\r] 空白符
\S [^\t\n\x0B\f\r] 非空白符
\w [a-zA-Z_0-9] 单词字符(字母,数字下划线)
\W [^a-zA-Z_0-9] 非单词字符

边界

常用边界匹配字符
| 字符 | 含义 |
| —- | ———- |
| ^ | 以xxx开始 |
| $ | 以xxx结束 |
| \b | 单词边界 |
| \B | 非单词边界 |

量词

字符 含义
? 出现零次或一次(最多出现一次)
+ 出现一次或多次(至少出现一次)
* 出现零次或多次(任意次)
{n} 出现n次
{n,m} 出现n到m次
{n,} 至少出现n次

JS正则贪婪模式与非贪婪模式

贪婪模式：尽可能多的匹配;
非贪婪模式：让正则表达式尽可能少的匹配，一旦成功匹配不在继续尝试，在量词后面加上？即可;

分组

使用( )可以达到分组的功能，使量词作用于分组;
使用|可以达到或的效果;
$符号表示分组内容;
