flexbox常用属性整理

发表于 2017-02-26 分类于 React Native

flexbox 是 Flexible Box 的缩写，即为弹性盒子布局，是 React Native 应用开发中最常用的内容，该布局可以简单快速地完成各种伸缩性的设计。

布局模型

  • 由伸缩容器和伸缩项目组成，任何一个元素都可以指定为 flexbox 布局，设为 display：flex 或 display：inline-flex 的元素称为伸缩容器。伸缩容器的子元素称为伸缩项目。

  • 默认情况下，伸缩容器由两个轴组成，即主轴（main axis）和交叉轴（cross axis），其中主轴的开始位置叫做 main start，结束位置叫 main end。交叉轴的开始位置叫 cross start，结束位置叫 cross end。

  • 伸缩项目在主轴上占据的空间叫 main size，在交叉轴上占据的空间叫 cross size。

  • 主轴既可以是水平轴，也可以是垂直轴。伸缩项目总是沿着主轴，从主轴开始位置到主轴结束位置进行排列。

伸缩容器属性

display

指定元素是否为伸缩容器，语法是：

  • flex:用于产生块级伸缩容器

  • inline-flex:用于产生行内级伸缩容器

flex-direction

指定主轴方向，语法是：

  • row:为默认值，伸缩容器为水平方向轴，伸缩项目排版方式从左向右排列

  • row-reverse:伸缩容器为水平方向轴，伸缩项目排版方式从右向左排列

  • column:伸缩容器为垂直方向轴，伸缩项目排版方式从上向下排列

  • column-reverse:伸缩容器为垂直方向轴，伸缩项目排版方式从下向上排列

flex-wrap

指定伸缩容器的主轴线方向空间不足的情况下，是否换行以及该如何换行，语法是：

  • nowrap:即使空间不足，伸缩容器也不允许换行

  • wrap:伸缩容器在空间不足的情况下允许换行。若主轴为水平轴，则换行的方向为从上到下

  • wrap-reverse:伸缩容器在空间不足的情况下允许换行。若主轴为水平轴，则换行的方向为从下到上

flex-flow

该属性是 flex-direction 和 flex-wrap 属性的简写版本,它同时定义了伸缩容器的主轴方向和换行方式，默认值为 row nowrap 语法是：

justify-content

该属性定义伸缩项目沿主轴线的对齐方式，语法是：

  • flex-start:伸缩项目向主轴线的起始位置靠齐

  • flex-end:伸缩项目向主轴线的结束位置靠齐

  • center:伸缩项目向主轴线的中间位置靠齐

  • space-between:伸缩项目会平均地分布在主轴线里。第一个伸缩项目在主轴线的开始位置，最后一个伸缩项目在主轴线的终点位置

  • space-around:伸缩项目会平均地分布在主轴线里，两端保留一半的空间

align-items

该属性用来定义伸缩项目在伸缩容器的交叉轴上的对齐方式，语法是：

  • flex-start:伸缩项目向交叉轴的起始位置靠齐

  • flex-end:伸缩项目向交叉轴的结束位置靠齐

  • center:伸缩项目向交叉轴的中间位置靠齐

  • baseline:伸缩项目根据它们的基线对齐

  • stretch:伸缩项目在交叉轴方向拉伸填充整个伸缩容器

align-content

该属性主要是调整伸缩项目出现换行后在交叉轴上的对齐方式，类似于伸缩项目在主轴上使用 justify-content,语法是：

  • stretch:默认值，伸缩项目在交叉轴上伸展以占用剩余空间。

  • flex-start:伸缩项目向交叉轴的起始位置靠齐

  • flex-end:伸缩项目向交叉轴的结束位置靠齐

  • center:伸缩项目向交叉轴的中间位置靠齐

  • space-between:伸缩项目在交叉轴平均分布

  • space-around:伸缩项目在交叉轴中平均分布，且在两端保留一半的空间

伸缩项目属性

order

定义项目的排列顺序，数值越小，排列越靠前，默认值为 0，语法：

flex-grow

定义伸缩项目的放大比例，默认值为 0，即如果存在剩余空间，也不放大。如果所有项目都设置为 1，则每个项目将设置为大小相等的剩余空间。如果将其中一个项目设置为 2，则这个项目所占剩余空间是其他项目所占剩余空间的两倍。语法是：

flex-shrink

定义伸缩项目的收缩能力，语法是：

flex-basis

用来设置伸缩项目的基准值，剩余空间按比率进行伸缩，语法是：

flex

该属性是 flex-grow、flex-shrink 和 flex-basis 这 3 个属性的缩写，默认值是 0 1 auto，其中第二和第三参数可选，语法是：

align-self

用来设置单独的伸缩项目在交叉轴上的对齐方式，会覆盖默认的对齐方式，语法是：

  • auto:伸缩项目按照自身设置的宽高显示，如果没有设置，则按 stretch 计算其值

  • flex-start:伸缩项目向交叉轴的开始位置靠齐

  • flex-end:伸缩项目向交叉轴的结束位置靠齐

  • center:伸缩项目向交叉轴的中心位置靠齐

  • baseline:伸缩项目按基线对齐

  • stretch:伸缩项目在交叉轴方向占满伸缩容器，在不设置高度的情况下才起作用
