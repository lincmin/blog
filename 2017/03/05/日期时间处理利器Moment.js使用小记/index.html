简介

  • version : 2.18.1
    Moment.js是一个JavaScript 日期处理类库。支持多语言，可以轻松处理开发中遇到的日期时间问题。在企业级后台产品中，对日期时间的处理是一个常见的需求，比如按照一定的格式显示日期时间、日期时间的加减计算、日期时间的前后比较等。以下记录了一些开发中接触到的常用方法。详细内容可浏览官网。
