安装node以及node常用工具

发表于 2017-11-20 分类于 node

这篇文章主要记录了 NodeJS 的安装方式以及 NodeJS 常用的管理工具，包括 NodeJS 版本管理工具 nvm，NodeJS 包管理工具 npm ， npm 源管理工具 nrm 以及 NodeJS 应用的进程管理器 pm2。

nvm

nvm 是 NodeJS 版本管理工具。可以让你在同一台机器上安装和切换不同的 NodeJS 版本。

NPM 简介

NPM 简介

NPM 是随同 NodeJS 一起安装的模块管理工具，利用 NPM 模块管理工具可以从 NPM 服务器下载第三方模块到本地使用，也可以将自己编写的模块上传至 NPM 服务器分享给别人使用。
